
This dataset contains 12,000 imagens from Leeds Sport Pose and its extension. The images were cropped arround the person with body joints annotations and resized to 220x220 with mean padding for mainteining the aspect ratio.

11,000 images were used for training and 1,000 for testing


For using the images download the compressed file from the link bellow and extract it to Datasets/Leeds_Extended

	link:	https://1drv.ms/u/s!AtZrNSTwd90_gpNluq6kFTxqTu_6AA

the locations for train and test sets are in the .npy files
